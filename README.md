# 🎓 StudyNotion - MERN Learning Platform

## 🚀 About the Project
StudyNotion is a full-stack MERN web application for online learning. It allows users to browse courses, enroll, and learn online.

---

## 🛠 Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Tailwind CSS
